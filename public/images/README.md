# Images Directory

Place your images in this directory. The application expects images such as:

- clouds.jpeg
- coins_investing.png
- running_silhouette.png
- knowledge.jpg
- reed.jpg
- mom-dad-wedding-resized.JPG
- Various podcast images (dotnetrocks.jpg, hanselminutes.jpg, etc.)

If images are not available, the application will use placeholder images.
